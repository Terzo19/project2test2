Main, Footer, Article and Section

Tag Name Description
<main> Main By using this element, web browsers are able to identify the that this will be the bulk of the content.
<footer> Footer This element is used to contain contact information,and is typically located at the bottom of the page.
<article> Article This element holds content that makes sense on its own such as articles,blogs, comments, magazines, etc.
<section> Section This element defines elements in a document such as chapters, headings or any other area with the same theme.

Header and Nav

Tag Name Description
<header> Header This tag is a container, usually either for navigational links or introductory content.
<nav> Navigation This tag is used to define a block of navigational links such as menus and tables of contents.
